# FoundrySpec CLI Footnote

The **FoundrySpec CLI** is the central heart of the documentation engine. It handles:

- **Initialization**: Scaffolding new projects from scratch.
- **Building**: Converting Mermaid diagrams and Markdown into a premium viewer experience.
- **Serving**: Providing a local development server with hot-reload capabilities.
- **Deployment**: Syncing the local state with GitHub Pages.

It is designed to be highly reliable and AI-friendly, enabling "Zero-Question Implementation".
